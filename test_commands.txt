# 実習4: n語長多倍精度加算のテストコマンド

# n = 3 (3語長 = 24ビット) を170H番地に設定
sm 170 03

# 第1数 = 0x123456 をLittle Endianで180H番地～に設定
sm 180 56
sm 181 34
sm 182 12

# 第2数 = 0x789ABC をLittle Endianで190H番地～に設定
sm 190 BC
sm 191 9A
sm 192 78

# 結果領域を0で初期化（1A0H番地～）
sm 1A0 00
sm 1A1 00
sm 1A2 00

# プログラムをロード
r test/multi_precision_binary.txt

# 実行前の状態確認
d
m 170
m 180
m 190
m 1A0

# プログラム実行
c

# 実行後の結果確認
d
m 1A0
m 1A1
m 1A2
